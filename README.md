#!!! Trzeba go umieścić w pierwszej linijce pliku main.js


Ten projekt ma na celu stworzenie prostej aplikacji wyświetlającej aktualną pogodę dla wybranego miasta na podstawie danych z API.

Funkcje:
- Wyszukiwanie miasta
- Ziana jednostki metrycznej
- Zmiana motywu (jasny/ciemny)
- Wyświetlanie temperatury
- Opis warunków pogodowych
- Wilgotność i prędkość wiatru
- Prognoza na dzień
- Prognoza na kilka dni

Napisane z pomocą:
HTML, CSS, JavaScript, API pogodowe (OpenWeather)
